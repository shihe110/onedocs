# redis入门之配置

# 配置

  • 1.命令配置
> CONFIG GET 配置名称

// 例如
> CONFIG GET loglevel

// 通配*获取所有配置
>   CONFIG GET *
  • 2.配置文件配置 redis.conf文件

# 配置语法

> CONFIG SET 配置项 配置值
// 例如
> CONFIG SET loglevel "notice"
